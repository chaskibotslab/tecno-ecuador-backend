Este es un ejemplo de frontend en React para probar la subida de imágenes a Google Drive y guardar los datos en Airtable usando tu backend.

1. Instala las dependencias:
   npm install

2. Ejecuta el frontend:
   npm start

3. Asegúrate de que tu backend esté corriendo en http://localhost:3001

Incluye un formulario para subir imagen y guardar datos en Airtable.
